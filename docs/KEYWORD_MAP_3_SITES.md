# KEYWORD MAP FOR SUPERPARTY, WOWPARTY, ANIMATOPIA

## 1. Anti-Cannibalization Policy
To prevent cross-site cannibalization:
- **Superparty.ro**: Broad, high-volume commercial intent (`animatori copii bucuresti`, `petreceri copii acasa`).
- **Animatopia.ro**: Premium, narrative, mascot-focused, interactive shows (`spectacol magie copii`, `inchiriere mascote disney`, `petreceri tematice premium`).
- **WowParty.ro**: Accessible, packages, quick-booking, budget-friendly (`pachete petreceri ieftine`, `animatori sector 3 pret`).

**Primary:**

 **Location:** Primary hub for Sector 1-6 & Ilfov (Commercial intent).
